# sync-skill

This skill helps keep your skills synced across different coding agents.

## Setup

1. Create an empty GitHub repo to use as your shared skills ledger.
2. Copy the link of that repo.
3. Give your coding agent this file and ask it to follow the instructions in `sync.md`.

Example:

```text
Follow instruction at this file: https://github.com/thedon-ai/sync-skill/blob/main/sync.md
```

When the agent asks for your ledger repo, paste the GitHub repo link you created in step 1.
